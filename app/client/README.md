# Movie Review Aggregator

## Client side

### How to run locally

1. Go into the client directory:

   ```sh
   cd app/client
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Create a file named `.env` following the format in `.env.template` file:

   ```xml
   NEXT_PUBLIC_SERVER_URL=http://localhost:5000
   ```

4. Start the client:

   ```sh
   npm run dev
   ```
